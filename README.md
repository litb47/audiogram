# Audiogram Labeler

A minimal web UI for expert labelers to review sanitized audiogram images and attach structured hearing-loss labels.

## Quick Start

```bash
cp .env.example .env.local
# Edit .env.local with your Supabase project URL and anon key
npm install
npm run dev
```

See [CLAUDE.md](./CLAUDE.md) for full setup including required Supabase SQL.
